# Fully Homomorphic Encryption (FHE) Research

A curated collection of FHE research resources, interactive tools, and learning materials for privacy-preserving computation.

---

## Overview

This repository hosts a web-based resource hub for Fully Homomorphic Encryption — covering core schemes, open-source libraries, seminal papers, and ML+FHE tooling.

## Live Pages

- **FHE Research Resources** — Comprehensive guide to FHE schemes, libraries, papers, and community links
- **Projects** — Portfolio of related research projects
- **Home** — Personal landing page

## Topics Covered

### FHE Schemes
- BGV, BFV, CKKS, TFHE, FHEW

### Libraries & Frameworks
- **OpenFHE** — Lattice-based cryptography library (Duality Technologies)
- **TenSEAL** — Privacy-preserving ML with homomorphic encryption
- **Concrete-ML** — ML framework over FHE (Zama)
- **HElib** — IBM's HE library
- **SEAL** — Microsoft's encryption library

### ML + FHE
- Encrypted inference and training
- Federated Learning with HE
- Privacy-preserving data analysis

### Key Papers
- Gentry (2009) — First FHE construction
- BGV, BFV, CKKS scheme papers
- TFHE bootstrapping
