FHE Libraries

Open-source libraries for implementing homomorphic encryption schemes, ranging from low-level cryptographic primitives to high-level ML-ready frameworks.

OpenFHE

Duality Technologies / OpenFHE Community

The successor to PALISADE, OpenFHE is a comprehensive open-source FHE library supporting all major schemes. It provides hardware acceleration, multi-party computation support, and a modular architecture for research and production use.

C++ Python bindings BSD 2-Clause
BGV BFV CKKS TFHE FHEW
GitHub Website Docs

Microsoft SEAL

Microsoft Research

One of the most widely-used FHE libraries, Microsoft SEAL provides efficient implementations of BFV and CKKS schemes. Known for its clean API design, extensive documentation, and strong community support.

C++ .NET / Python wrappers MIT
BFV CKKS BGV
GitHub Website

HElib

IBM Research

One of the earliest FHE libraries, HElib implements the BGV scheme with advanced optimizations including bootstrapping, smart automorphisms, and the GHS variant for efficient SIMD-style operations on encrypted data.

C++ Apache 2.0
BGV CKKS
GitHub Docs

TFHE-rs

Zama

A pure Rust implementation of the TFHE scheme, designed for boolean and integer FHE computations. TFHE-rs is the core engine behind Zama's Concrete framework and supports programmable bootstrapping for arbitrary function evaluation.

Rust C / Python bindings BSD 3-Clause
TFHE
GitHub Docs

Concrete

Zama

A Python framework that compiles regular Python programs into FHE equivalents. Concrete allows developers to write standard NumPy code and automatically converts it into FHE circuits, dramatically lowering the barrier to using FHE.

Python BSD 3-Clause
TFHE
GitHub Docs

Concrete-ML

Zama

Built on top of Concrete, Concrete-ML provides privacy-preserving ML tools with APIs compatible with scikit-learn, PyTorch, and XGBoost. It handles quantization-aware training and FHE compilation automatically.

Python BSD 3-Clause

GitHub Docs

TenSEAL

OpenMined

A Python library for performing tensor operations on encrypted data, built on top of Microsoft SEAL. TenSEAL provides a PyTorch-like API for privacy-preserving deep learning, making it easy to run neural network inference on encrypted data.

A Go library for lattice-based multiparty homomorphic encryption. Lattigo stands out with its native multiparty HE support and efficient implementation of RLWE-based schemes, designed for cloud-native and distributed settings.

FHE Schemes

Understanding the major homomorphic encryption schemes, their properties, and ideal use cases.

BGV

Brakerski-Gentry-Vaikuntanathan

An integer-arithmetic leveled HE scheme based on the Ring-LWE problem. Efficient for exact computations on integers with noise management via modulus switching.

  • Exact integer arithmetic
  • SIMD batching via CRT packing
  • Leveled (depth-limited without bootstrapping)
  • Best for: integer computations, database queries

BFV

Brakerski/Fan-Vercauteren

Similar to BGV but with a different noise management strategy (scale-invariant). Provides exact integer arithmetic with simpler parameter selection.

  • Scale-invariant noise management
  • Exact integer arithmetic
  • Simpler parameter tuning than BGV
  • Best for: integer workloads, simple circuits

CKKS

Cheon-Kim-Kim-Song

Supports approximate arithmetic on real and complex numbers. The go-to scheme for machine learning on encrypted data since ML workloads are naturally tolerant to small approximation errors.

  • Approximate real/complex arithmetic
  • Efficient for floating-point workloads
  • Rescaling for noise management
  • Best for: ML inference, statistical analysis

TFHE

Torus Fully Homomorphic Encryption

Operates on individual bits (or small integers) with fast bootstrapping. The programmable bootstrapping feature allows evaluation of arbitrary lookup tables during the bootstrapping operation.

  • Fast gate-by-gate bootstrapping
  • Programmable bootstrapping (PBS)
  • Boolean and small-integer circuits
  • Best for: arbitrary functions, control flow

FHEW

Fully Homomorphic Encryption over the torus (variant)

A predecessor to TFHE, FHEW introduced the concept of fast bootstrapping for binary gates. It achieves sub-second bootstrapping times, making it practical for boolean circuit evaluation.

  • Fast boolean gate bootstrapping
  • Small ciphertext sizes
  • Foundation for TFHE improvements
  • Best for: boolean circuits

GSW

Gentry-Sahai-Waters

An asymmetric noise growth scheme that simplifies homomorphic multiplication by avoiding key-switching and modulus-switching. Used as a building block in many modern FHE constructions.

  • Asymmetric noise growth
  • Simpler multiplication procedure
  • Building block for other schemes
  • Best for: theoretical constructions

Key Research Papers

Foundational and landmark papers in fully homomorphic encryption, from Gentry's breakthrough to modern applications in machine learning.

Foundational

Fully Homomorphic Encryption Using Ideal Lattices

Craig Gentry

STOC 2009

The seminal paper that introduced the first plausible construction of a fully homomorphic encryption scheme, based on ideal lattices with a bootstrapping technique to refresh ciphertexts.

Foundational

Fully Homomorphic Encryption without Bootstrapping

Zvika Brakerski, Craig Gentry, Vinod Vaikuntanathan

ITCS 2012

Introduced the BGV scheme with modulus-switching technique, enabling leveled FHE without bootstrapping for bounded-depth circuits, dramatically improving practical efficiency.

Foundational

Somewhat Practical Fully Homomorphic Encryption

Junfeng Fan, Frederik Vercauteren

IACR ePrint 2012

Introduced the BFV (Fan-Vercauteren) scheme, a scale-invariant variant of Brakerski's scheme that simplifies parameter selection and implementation.

Foundational

Homomorphic Encryption for Arithmetic of Approximate Numbers

Jung Hee Cheon, Andrey Kim, Miran Kim, Yongsoo Song

ASIACRYPT 2017

Introduced the CKKS scheme for approximate arithmetic on encrypted real numbers, enabling practical FHE for machine learning and data analytics applications.

Foundational

TFHE: Fast Fully Homomorphic Encryption over the Torus

Ilaria Chillotti, Nicolas Gama, Mariya Georgieva, Malika Izabachene

Journal of Cryptology 2020

Presented the TFHE scheme with fast gate bootstrapping (under 10ms per gate), enabling practical evaluation of boolean circuits on encrypted data.

Survey

A Survey on Homomorphic Encryption Schemes: Theory and Implementation

Abbas Acar, Hidayet Aksu, A. Selcuk Uluagac, Mauro Conti

ACM Computing Surveys, 2018

Comprehensive survey covering the theoretical foundations and practical implementations of HE schemes across multiple generations, with performance comparisons.

Survey

SoK: Fully Homomorphic Encryption Compilers

Alexander Viand, Patrick Jattke, Anwar Hithnawi

IEEE S&P 2021

Systematization of knowledge on FHE compilers, comparing different approaches to making FHE accessible through automatic compilation and optimization.

Application

CryptoNets: Applying Neural Networks to Encrypted Data

Ran Gilad-Bachrach, Nathan Dowlin, et al.

ICML 2016

Demonstrated the feasibility of applying trained neural networks to HE-encrypted data, achieving 99% accuracy on encrypted MNIST with practical throughput.

Application

GAZELLE: A Low Latency Framework for Secure Neural Network Inference

Chiraag Juvekar, Vinod Vaikuntanathan, Anantha Chandrakasan

USENIX Security 2018

Combined HE with garbled circuits for efficient private neural network inference, achieving orders of magnitude improvement over prior work.

Application

Programmable Bootstrapping Enables Efficient Homomorphic Inference of Deep Neural Networks

Jordan Frery, Andrei Stoian, et al. (Zama)

IACR ePrint 2023

Demonstrated how TFHE's programmable bootstrapping enables efficient deep neural network inference on encrypted data, powering the Concrete-ML framework.

Suggested Learning Path

A recommended progression for learning FHE, from cryptographic foundations to building privacy-preserving ML applications.

1. Foundations

Start here

  • Learn lattice-based cryptography basics (LWE, RLWE)
  • Understand noise growth and modular arithmetic
  • Read Gentry's thesis overview or Halevi's tutorial
  • Study the HomomorphicEncryption.org security standard

2. Core Schemes

Understand the schemes

  • Study BFV/BGV for exact integer arithmetic
  • Learn CKKS for approximate arithmetic (key for ML)
  • Explore TFHE for boolean/integer operations
  • Understand trade-offs between schemes

3. Hands-on Libraries

Build with libraries

  • Start with Microsoft SEAL examples (well-documented)
  • Explore OpenFHE for multi-scheme support
  • Try TenSEAL for Python-friendly HE operations
  • Experiment with Concrete-ML for ML workflows

4. ML Applications

Privacy-preserving ML

  • Implement encrypted inference with TenSEAL
  • Build FHE-compatible models with Concrete-ML
  • Combine FHE with federated learning (CKKS + FL)
  • Benchmark accuracy vs. encryption overhead
